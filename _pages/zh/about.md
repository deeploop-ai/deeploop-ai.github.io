DeepLoop AI Labs 是一家 AI 工程实验室，连接开源创新与企业级交付。我们构建驱动 Agent 原生应用的基础设施，与企业合作定制 AI 开发，并培训工程团队交付生产级 AI 系统。

## 三大方向

### 开源项目

我们在开源社区开发并维护 Agent 原生工具：

- **[Memoss](https://github.com/deeploop-ai/memoss)** — Agent 原生知识运行时，LLM Agent 持续从数据源构建并维护活的知识库。
- **[Graviton](https://github.com/deeploop-ai/graviton)** — 基于 Go 的 AI 原生 BaaS 平台，提供身份认证、动态数据库、文件存储和函数执行。
- **[Bluepilot](https://github.com/deeploop-ai/bluepilot)** — 编写 Markdown 规格，确定性生成交互式 SvelteKit 原型，加速设计评审。
- **[EventR](https://github.com/deeploop-ai/eventr)** — 基于 Go 的 DAG 事件路由器，通过 Source → Transform → Sink 拓扑处理流式数据，支持条件路由与至少一次交付。

### AI 项目外包开发

我们为企业提供端到端 AI 解决方案——从 LLM Agent 系统、RAG 管道到计算机视觉和预测分析。工程师团队融入你的技术栈，交付生产就绪的系统，并内置知识转移。

### AI 培训

我们举办实战工作坊和训练营，涵盖 LLM 应用开发、Agent 架构、Prompt 工程及 MLOps。培训面向希望构建而非仅消费 AI 能力的工程团队。

## 我们的方式

我们以真正的合作伙伴身份协作。每个项目都从理解你的独特挑战开始，每次交付都包含团队持续前进所需的文档和培训。我们以你能独立运行的系统来衡量成功——而非仅看工时与交付物。
